# cypress-examples

This project is a sample using Cypress on [BlazeMeter demo site](http://demo.blazemeter.com).

# setup

Run using docker compose:

```bash
docker-compose up -c
docker-compose exec cypress bash
```

Install the dependencies:

```bash
npm install
```

Run the tests:

```bash
npm test
```
